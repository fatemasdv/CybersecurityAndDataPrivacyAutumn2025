## 1. About This Policy
This Privacy Policy explains how the Booking System collects, uses, and protects personal data in compliance with the GDPR.

## 2. Personal Data We Collect
We collect only the personal data necessary for system operation and booking management:
- Email address (used as username)
- User role (reserver or administrator)
- Birthdate (to verify that users are over 15 years old)
- Reservation details (resource and time)
- Login IP address and time
- Session and security tokens

## 3. Purpose of Data Processing
We use personal data to:
- Authenticate users and allow secure login
- Manage bookings and resources efficiently
- Verify age eligibility
- Protect the system from misuse or unauthorized access

## 4. User Consent
By registering, users provide explicit consent for their personal data to be processed in accordance with this policy.

## 5. Data Protection Measures
- Passwords are securely hashed before storage
- CSRF protection is implemented to prevent attacks
- Access to data and system functionality is controlled by user roles

## 6. Cookies
The system uses essential cookies to maintain login sessions and enhance security. These cookies are required for proper system operation.

## 7. User Rights
Users have the right to:
- View and update their personal information
- Request deletion of their account and personal data
- Withdraw consent for data processing by contacting the administrator

## 8. Data Sharing
Personal data is **not shared with any third parties**.

## 9. Policy Updates
This policy may be updated when necessary. Users will be notified of any significant changes.

## 10. Contact
For questions or concerns regarding privacy, users can contact the system administrator.
