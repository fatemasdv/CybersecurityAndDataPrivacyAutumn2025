# GDPR Compliance Checklist

| Category | Status | Verification |
| :--- | :---: | :--- |
| Data Minimization | ✅ | Only necessary data (email, age, role) is collected. |
| User Consent | ✅ | Consent is obtained via the Terms of Service checkbox during registration. |
| Data Security | ✅ | Bcrypt hashing for passwords and CSRF tokens for sessions. |
| User Rights | ✅ | Users can request to see all personal data the system holds about them, such as their profile details and booking history. |
| Public Privacy | ✅ | Public users cannot see the identities of reservers. |
| Data Breach Plan |  ❌  | Documented procedure for handling security incidents is now included. |
